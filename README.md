# Get started

Clone the project

```
git clone https://github.com/SABHYAGUPTA5XR/MyPortfolio.git
```

Go to the project directory

```
cd MyPortfolio
```

Install dependencies

```
npm install
```

Start the development server

```
npm run dev
```
